# TaskFlow – konsolowy menedżer zadań w Pythonie

Projekt zaliczeniowy - języki programowania.

## Opis projektu
TaskFlow to aplikacja konsolowa napisana w języku Python, służąca do zarządzania zadaniami (To-Do List).
Umożliwia:
- dodawanie zadań
- usuwanie zadań
- oznaczanie zadań jako wykonane
- sortowanie zadań według priorytetu (algorytm Merge Sort)

Dane są przechowywane w lokalnej bazie danych SQLite, co umożliwia trwałe zapisywanie informacji.

## Zastosowane technologie
- Python
- Jupyter Notebook
- SQLite
